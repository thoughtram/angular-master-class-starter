Contacts Server

REST API documentation for the Contacts HTTP server (used in thoughtram AMC).

  • getContacts() /api/contacts GET
  • getContact(id) /api/contacts/:id GET
  • searchContacts(query) /api/search?text==<xxx> GET
  • checkEmail(email) /api/check-email?email=<xxx> GET
  • createContact(contact) /api/contacts/:id PUT
  • updateContact(contact) /api/contacts PUT
  • deleteContact(id) /api/contacts/:id DELETE
